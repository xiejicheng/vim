## Vim 模式

#### Vim 存在多种模式

- vim 默认的是 normal (普通) 模式
- 使用 a 、i 等进入编辑模式
- :cmd 命令模式
- v(visual) 可视化模式

#### vim 默认模式是 normal 模式

- 按 Esc 键可以从插入模式回到 normal 模式
- 普通模式可以进行命令操作和移动操作

#### Insert(插入) 模式

- 使用 i 、a 、o 进入插入模式
- 使用 Esc 返回到普通模式

#### Command(命令)模式

- 就是在 vim 下执行命令
- 保存 :w
- 退出 :q
- 分屏命令 :vs(vertical split)垂直分屏| ；:sp(split)水平分屏—
- 全局替换 :% s/foo/bar/g

#### Visual (可视)模式

- normal 模式下使用 v 进入visual 选择
- 使用 V 选择行
- 使用 ctrl + v 进行方块选择
