## Vim 与 Git联手

### vim-fugitive

一个 Git 包装器

- [地址](https://github.com/tpope/vim-fugitive)

- 命令：Gedit, Gdiff, Gblame, Gcommit等



### vim-gitgutter

在 vim 里显示文件变动

- [地址](https://github.com/airblade/vim-gitgutter)

- 修改文件之后可以显示当前文件的变动
- 哪些行新增，哪些行修改了，哪些行删除了



### gv.vim

在命令行查看提交记录

- [地址](https://github.com/junegunn/gv.vim)

- 使用命令 `:GV`  调用
- 可以浏览代码提交变更
